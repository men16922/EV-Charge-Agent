**형식:** Mac 화면녹화(≤10초), LinkedIn에 **네이티브 영상**으로 업로드(YouTube ❌ — 네이티브가 피드에서 자동재생).

## 10초 샷 구성 (와우 포인트 하나: "물어보면 → 알아서 결정 → 경로")

**녹화 전 세팅 (그래야 클립이 군더더기 없이 임팩트만):**
1. 로컬(localhost) 앱을 풀스크린·100% 줌·새로고침. **주소창은 녹화에서 제외**(앱 영역만).
2. 도시 **서울** 선택 → IONIQ 5 자동 로드. 배터리 슬라이더 **20%**.
3. 지도를 차 마커 중심으로, 충전 핀이 보이게. 아직 추천받기 누르지 말 것.

**이렇게 녹화 (~9초):**
- **0–2초:** 지도 위 IONIQ 5 차 마커 + 색색의 충전 핀 (시각적 훅)
- **2–3초:** 커서로 **추천받기** 클릭
- **3–7초:** BEST 카드 등장(🟢 빈자리 · 🔌 내 차 호환 · ✅ 도착 가능) **+ 파란 경로선이 지도에 그려짐**
- **7–9초:** 경로 + BEST 충전소 상태로 잠깐 정지. 끝.

> 🧠 **에이전트 추론 트레이스**가 킬러 기능이지만 LLM 응답에 몇 초 걸린다. 10초 티저에선 **추천→카드+경로**(즉시 반응)가 안정적인 한 방. 트레이스를 꼭 넣고 싶으면 조금 더 길게 찍고 편집에서 **배속(speed-ramp)** 으로 ≤10초에 맞춘다.

## Mac 녹화 방법
1. **Cmd + Shift + 5** → **선택 영역 기록** → 앱 주위로 타이트하게 박스 드래그(4:5 권장).
2. **기록** 클릭 → 위 샷 수행 → 메뉴바에서 정지.
3. 트리밍: `.mov`를 **QuickTime**에서 열고 **편집 ▸ 다듬기**로 ≤10초.
4. (선택) LinkedIn용 `.mp4` 변환·압축 — QuickTime **파일 ▸ 내보내기 ▸ 1080p**, 또는 `ffmpeg -i in.mov -vf scale=1080:-2 -crf 23 teaser.mp4`.
5. 소리 불필요 → **한 줄 자막 오버레이**로 무음에서도 읽히게.

**LinkedIn 사양:** mp4/mov, ≤10초, 1080p. **4:5(세로형) 또는 1:1이 모바일 피드를 더 꽉 채움** → 가능하면 4:5. 자동재생이 무음이라 자막/오버레이 필수.
